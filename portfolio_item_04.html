Unity 3DFirst Person Shooter

Metroid Survive

I created this first person shooter game using the Unity game engine and the C# programming language. This game is meant to showcase my skills in design and development in a first person shooter game.

I began this project by creating a basic weapon swapping system. The player has four different weapon types, all with different ranges, effects, and animations.

I then felt the game needed some sense of urgency to be completed. I added a mechanic where the players flashlight will dwindle over time and left power ups to recharge the flashlight hidden around the map.

Finally added a break in the players visor everytime they take damage. This gave the enemies attacks some impact and gave the player feedback. I also added a death screen letting the player know they are dead and giving them the ability to restart the level.
