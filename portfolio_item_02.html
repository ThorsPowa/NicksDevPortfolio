Slime Hunter

I created this website using JavaScript, HTML, and CSS. This website displays a fun card game where you have to match two of-a-kind cards in order to get a match. With six matches you win the game!

I began this site with a basic images for the front and backside of the cards. I then used JavaScript to give the game some functionality.

The cards are automatically randomized everytime the page is refreshed. When you select two different the JavaScriptcode alerts you and flips the cards back over so you can begin your selection again.

Finally if you do indeed select two of the same card the JavaScript alerts you that you have selected a pair, the cards then disapear and you can continue the game. Once all the cards are gone the script then alerts you that you have won!
